### Process
== Process ==  is just a running program
- It has Memory, a stack given by os, and cpu instructions in its == address space ==
- It has different state given to it by os 
  1. Ready
  2. Running
  3. Blocked
  4. Done 
- Process scheduling is done by OS and cpu is just a dumb executer
- Process list is list of all process right now cpu will execute 
- Each entry is just == PCB == which is simply a data structure to represent the == PROCESS ==
